DecredBot
==========

This is stunning Decred (DCR) IRC bot.

Usage
=====

Install ruby: `apt-get install ruby2.0`

Install bundler: `gem install bundler`

Inside cloned directory run: `bundle install` to install dependencies

Run bot: `ruby bot.rb`
